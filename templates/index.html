The D&D μStarter Kit is a helper tool for players of Dungeons and Dragons. Roll dice, generate a character with an ad-libbed one-line backstory, or play a complete ad-libbed D&D scene!

This kit makes use of quantum computing for randomness and ad-libs, alongside user inputs.

All generators are based on the D&D 5E Player’s Handbook.

For an immersive experience we recommend you listen to '2 Hours of Fantasy Music' by Adrian von Ziegler

For more information, please look at the README or check out the Github repo.
